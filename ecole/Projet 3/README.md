# Projet 4 – Analyse de vulnérabilités (OWASP Juice Shop)

## Objectif
Analyser plusieurs failles de sécurité sur l’application volontairement vulnérable OWASP Juice Shop dans un cadre pédagogique.

## Compétences travaillées
- Analyse de vulnérabilités web  
- Compréhension des failles OWASP  
- Manipulation d’API et DevTools  
- Rédaction de rapports techniques  

## Contenu du dossier
Le dossier contient plusieurs rapports, chacun décrivant :
- la vulnérabilité exploitée  
- la méthodologie  
- les risques  
- les correctifs recommandés  

## Conclusion
Ce projet m’a permis de comprendre et d’appliquer les bases de la cybersécurité web, notamment l’analyse et la correction de failles courantes.
